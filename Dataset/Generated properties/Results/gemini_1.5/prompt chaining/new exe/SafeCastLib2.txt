## Test Invariants for SafeCastLib

This document outlines test invariants for the `SafeCastLib` contract, focusing on edge cases and ensuring the contract's robustness.

**General Invariants:**

- **Invariant 1: Overflow Revert:** Any casting operation exceeding the target type's maximum value MUST revert with the `Overflow` error.
- **Invariant 2: Underflow Revert:** Any casting operation resulting in a value below the target type's minimum value MUST revert with the `Overflow` error.
- **Invariant 3: Successful Conversion:** For input values within the target type's range, the casting operation MUST return the correctly converted value.
